예를 들어, 다른사람의 코드를 읽는 능력, 다른 직군과의 소통 방법, 시간이 생명인 운영업무를 통한 업무 일정 조율 및 속도 향상, 언제든지 담당자가 바뀔 수 있는 운영 업무의 특성을 고려한 페이지 특성에 맞게 코드 작성 스타일 바꾸기, JSP 해석이 필요하여 JSP 독학을 통한 JSP 소스 해석등이 있습니다.
웹 접근성 심사에서 열심히 노력하여 100점으로 인증마크를 취득하는 쾌거를 이룰 수 있었습니다.
주로 하였던 업무는 이벤트 페이지 제작, 메인 페이지 제작, jsp 페이지 수정, 이벤트 및 공지 게시물 제작, 팝업 제작, 이메일링, 신규 페이지 제작등이 있습니다.
운영 도중, 타사에서 리뉴얼을 하게되어 전체 페이지가 변함으로, 업무 스타일이 변하는 이슈가 있었으나, 타사와의 소통을 통해 금방 적응 할 수 있었습니다.
